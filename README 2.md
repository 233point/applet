## 部署说明

### 安装环境
```
# 安装（更新） wepy 命令行工具。
npm install wepy-cli -g
# 安装依赖包
npm install
# 开发实时编译。
wepy build --watch
```

### 小程序工具说明
- 使用微信开发者工具-->添加项目，选择dist目录；
- 请修改APPID为自己的ID；
- 关闭ES6转ES5；
- 关闭代码压缩上传；

> [WePY开发文档参考](https://tencent.github.io/wepy/)
> [小程序云开发文档参考](https://tencent.github.io/wepy/)
